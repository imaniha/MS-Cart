# CART API

## SoapUI

### Installation

```
cd /opt
wget http://cdn01.downloads.smartbear.com/soapui/5.2.1/SoapUI-x64-5.2.1.sh
sh SoapUI-x64-5.2.1.sh
```

### Lancement de SoapUI en mode graphique

```
/opt/SoapUI-5.2.1/bin/soapui.sh &
```

### Lancement des tests depuis le projet

```
sh bin/soapui.sh
```
